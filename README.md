# Python Weekly Assignments Repository

## Overview
This repository contains weekly Python programming assignments focused on mathematical and algorithmic problems, helping build foundational and advanced programming skills.

## Topics Covered
- Factorials, palindrome checks, digit operations
- Number theory: prime factors, abundant/deficient numbers, Harshad numbers, automorphic and pronic numbers
- Prime computations: Mersenne primes, twin primes, prime power checks
- Modular arithmetic and cryptographic concepts
- Advanced algorithms: Miller-Rabin test, Pollard's rho factorization, quadratic residues
- Additional mathematical sequences and functions

## How to Use
- Clone the repo
- Explore week-wise folders containing scripts
- Run the Python scripts with Python 3.x
- Test functions with sample inputs provided

## Skills Gained
- Python programming for math problems
- Efficient algorithm implementation
- Understanding of number theory and modular arithmetic
